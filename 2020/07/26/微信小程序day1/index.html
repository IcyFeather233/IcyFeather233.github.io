项目结构

pages

存放页面的文件夹，每一个子文件夹对应一个页面

例如

app.js

应用入口文件

app.json

应用配置文件

app.wxss

全局样式文件

组件

基本组件

view

布局标签，相当于div，块级元素

text

文本标签，相当于span，行级元素

表单组件

form

表单

button

  • 按钮
  • 属性
    • type
      • primary 绿色
      • warn 红色
      • default 白色
    • size
      • default
      • mini

input

  • 文本输入框
  • 属性 type, 默认为text

radio

单选框，一般外面需要嵌套radio-group组件

checkbox

复选框，一般外面需要嵌套checkbox-group组件

textarea

文本域

switch

开关组件，checked属性

导航组件

navigator

  • 普通页面的跳转
  • 属性 url 指定目标页面地址

image

图片组件

  • src
  • lazy-load

audio

音频组件

  • src
  • cover
  • controls

video

视频组件

  • src
  • cover
  • controls
