如何在hexo上写博客

2020-07-20

首先进入到hexo所在的目录
hexo s 本地localhost:4000访问页面 ( 需要hexo g之后才能执行 )
hexo clean + hexo g + hexo d 可以上传到github个人主页

hexo g 类似编译命令
hexo g 中 g = generate
hexo d 中 d = deploy
